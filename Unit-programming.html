Chapter 1: Computer Systems & Programming Fundamentals

A computer is an electronic device that processes raw data into useful information[span_0](start_span)[span_0](end_span). It relies on a symbiosis between hardware (physical components) and software (the programs that control them)[span_1](start_span)[span_1](end_span).

The Programming Hierarchy

Languages are categorized by how close they are to the hardware[span_2](start_span)[span_2](end_span):

  • Low-Level: Machine/Assembly (Direct hardware control)[span_3](start_span)[span_3](end_span).
  • High-Level: C, Java, Python (Human-readable, requires translation)[span_4](start_span)[span_4](end_span).

Chapter 4: Control Structures & Iteration
