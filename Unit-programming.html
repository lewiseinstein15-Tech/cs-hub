1. Introduction to Computer Science

Computer Science is the scientific study of computers, computational systems, software, algorithms and information processing. It combines mathematics, logic, engineering and programming to solve real-world problems using technology.

Today, Computer Science powers smartphones, artificial intelligence, cybersecurity, robotics, cloud computing, data science, software engineering and almost every modern industry.

2. Evolution of Computers

Computers have evolved over thousands of years, from simple counting devices to powerful electronic systems capable of performing billions of calculations every second.

2.1 Early Calculating Devices

Abacus (Around 3000 BC)

The abacus is one of the earliest known calculating devices. It uses beads that slide along rods to perform arithmetic operations such as addition, subtraction, multiplication and division.

Advantages

  • Simple to use.
  • Does not require electricity.
  • Fast for basic arithmetic calculations.

Napier's Bones (1617)

Invented by John Napier, this mechanical device used numbered rods to simplify multiplication and division.

Pascaline (1642)

Invented by Blaise Pascal, the Pascaline was one of the first mechanical calculators capable of performing addition and subtraction.

Leibniz Calculator (1673)

Developed by Gottfried Wilhelm Leibniz, this calculator improved upon the Pascaline by performing multiplication and division more efficiently.

Difference Engine (1822)

Designed by Charles Babbage, the Difference Engine was built to calculate mathematical tables automatically. Charles Babbage is widely known as the Father of the Computer.

Analytical Engine (1837)

Charles Babbage later designed the Analytical Engine, considered the first concept of a general-purpose programmable computer. It introduced ideas similar to the CPU, memory, input and output used in modern computers.

Ada Lovelace

Ada Lovelace wrote the first algorithm intended for the Analytical Engine and is widely recognized as the world's first computer programmer.

2.2 Generations of Computers

Generation Period Main Technology Main Characteristics
First 1940–1956 Vacuum Tubes Very large, expensive and produced a lot of heat.
Second 1956–1963 Transistors Smaller, faster and more reliable.
Third 1964–1971 Integrated Circuits (ICs) Higher processing speed and lower power consumption.
Fourth 1971–Present Microprocessors Personal computers, laptops, smartphones and modern computing.
Fifth Present and Beyond Artificial Intelligence Machine learning, robotics, expert systems and natural language processing.

3. Characteristics of Computers

A computer possesses several unique characteristics that make it an essential tool in education, business, healthcare, engineering, banking, research, and many other fields.

1. Speed

Computers perform millions or even billions of operations every second. Tasks that would take humans hours or days can be completed within seconds.

2. Accuracy

A computer produces highly accurate results when given the correct data and instructions. Errors usually occur because of incorrect input or faulty programs rather than the computer itself.

3. Diligence

Unlike humans, computers do not become tired, bored, or lose concentration. They can perform the same task repeatedly with consistent performance.

4. Reliability

Computers can operate continuously for long periods while maintaining consistent and dependable results.

5. Storage Capacity

Modern computers can store vast amounts of information and retrieve it quickly whenever needed.

6. Automation

Once a program is started, a computer can complete tasks automatically without constant human intervention.

7. Versatility

A single computer can perform many different tasks, such as programming, gaming, graphic design, scientific simulations, video editing, and web browsing.

Key Points to Remember
  • Speed
  • Accuracy
  • Diligence
  • Reliability
  • Storage Capacity
  • Automation
  • Versatility

5. Computer Hardware and Software

A computer system consists of two major components: hardware and software. Both must work together for a computer to function properly.

5.1 Computer Hardware

Hardware refers to the physical parts of a computer that can be seen and touched.

Examples of Hardware

  • Keyboard
  • Mouse
  • Monitor
  • Central Processing Unit (CPU)
  • Motherboard
  • Hard Disk Drive (HDD)
  • Solid State Drive (SSD)
  • Printer
  • Scanner
  • Speakers

Categories of Hardware

Category Examples
Input Devices Keyboard, Mouse, Scanner, Microphone, Webcam
Output Devices Monitor, Printer, Projector, Speakers
Processing Devices CPU, GPU
Storage Devices Hard Disk, SSD, Flash Disk, Memory Card, DVD
Communication Devices Network Card, Modem, Router

5.2 Computer Software

Software is a collection of programs and instructions that tell the computer what to do. Unlike hardware, software cannot be touched physically.

Types of Software

1. System Software

System software manages the computer hardware and provides a platform for application software to run.

Examples:

  • Windows
  • Linux
  • macOS
  • Android
  • Device Drivers
  • Utility Programs

2. Application Software

Application software helps users perform specific tasks.

Examples:

  • Microsoft Word
  • Microsoft Excel
  • Google Chrome
  • Visual Studio Code
  • Adobe Photoshop
  • VLC Media Player
Exam Tip

A common examination question is:

Differentiate between computer hardware and computer software, giving five examples of each.

6. Types of Computers

Computers can be classified according to their size, processing power and purpose. Each type is designed to meet specific computing needs.

6.1 Classification by Size

Type Description Examples
Supercomputer The fastest and most powerful computers used for scientific research and complex simulations. Weather forecasting, space exploration, artificial intelligence.
Mainframe Computer Large computers capable of serving thousands of users simultaneously. Banks, airlines, government institutions.
Minicomputer Medium-sized computers that support multiple users. Universities and medium-sized organizations.
Microcomputer The most common type of computer designed for one user. Desktop computers, laptops, tablets and smartphones.

6.2 Classification by Purpose

  • General Purpose Computers – Perform many different tasks such as browsing the internet, programming, gaming and document processing.
  • Special Purpose Computers – Designed to perform one specific task, for example ATMs, traffic control systems and washing machine controllers.

7. Data and Information

Although the terms data and information are often used together, they have different meanings.

Data Information
Raw facts and figures that have not been processed. Processed data that is meaningful and useful.
Cannot support decision making directly. Can be used to make decisions.
Example: 75, 80, 90 Example: The student's average score is 82%.

Data Processing Cycle

  1. Data Collection
  2. Input
  3. Processing
  4. Storage
  5. Output
  6. Feedback
Key Points to Remember
  • Data = Raw facts.
  • Information = Processed data.
  • Information is produced after data has been processed.

8. Number Systems

A number system is a method of representing numbers using a specific set of digits. Computers operate using the binary number system because electronic circuits have only two stable states: ON and OFF.

Common Number Systems

Number System Base Digits Used
Binary 2 0, 1
Octal 8 0 – 7
Decimal 10 0 – 9
Hexadecimal 16 0 – 9, A – F

Why Computers Use Binary

  • Electronic circuits have only two stable states: ON and OFF.
  • Binary is reliable and easy for digital circuits to process.
  • Binary reduces errors in electronic communication.

Examples

Decimal Binary
2 10
5 101
8 1000
10 1010

11. Algorithms

An algorithm is a finite sequence of clear, logical and ordered steps used to solve a problem or perform a task.

Characteristics of an Algorithm

  • Clearly defined inputs.
  • Produces one or more outputs.
  • Steps are unambiguous.
  • Terminates after a finite number of steps.
  • Efficient and easy to understand.

Example Algorithm

Problem: Find the sum of two numbers.

  1. Start
  2. Input the first number.
  3. Input the second number.
  4. Add the two numbers.
  5. Display the result.
  6. Stop.
Exam Tip A common exam question is:
Define an algorithm and describe the characteristics of a good algorithm.

12. Flowcharts

A flowchart is a graphical representation of an algorithm. It uses standardized symbols connected by arrows to illustrate the sequence of operations required to solve a problem.

Common Flowchart Symbols

Symbol Name Purpose
Oval Start / End Shows the beginning or end of a program.
Rectangle Process Represents a calculation or instruction.
Parallelogram Input / Output Used for entering or displaying data.
Diamond Decision Represents a Yes/No or True/False decision.
Arrow Flow Line Shows the direction of execution.

Advantages of Flowcharts

  • Easy to understand.
  • Improves communication among programmers.
  • Simplifies debugging.
  • Acts as program documentation.
  • Makes program planning easier.

13. Introduction to C Programming

C is a general-purpose programming language developed by Dennis Ritchie at Bell Laboratories in 1972. It is one of the most influential programming languages and forms the foundation of many modern languages such as C++, Java and C#.

Features of the C Language

  • Fast and efficient.
  • Portable across different computer systems.
  • Supports structured programming.
  • Provides low-level memory access.
  • Widely used in operating systems and embedded systems.

Structure of a Simple C Program


#include <stdio.h>

int main()
{
    printf("Hello, World!");

    return 0;
}

Explanation

  • #include <stdio.h> imports the standard input/output library.
  • main() is the entry point of every C program.
  • printf() displays output on the screen.
  • return 0; indicates that the program ended successfully.
Remember

Every C program begins execution in the main() function.

14. Variables and Data Types

A variable is a named memory location used to store data. The value of a variable can change while a program is running.

Rules for Naming Variables

  • Must begin with a letter or an underscore (_).
  • Cannot begin with a number.
  • Cannot contain spaces.
  • Cannot use C reserved keywords (such as int, float, or return).
  • Should have meaningful names such as studentAge or totalMarks.

Common C Data Types

Data Type Description Example
int Stores whole numbers. 25
float Stores decimal numbers. 12.75
double Stores larger decimal values with higher precision. 12345.6789
char Stores a single character. 'A'

16. Programming Errors

Programming errors (bugs) are mistakes that prevent a program from working correctly.

Error Type Description
Syntax Error Violates the rules of the programming language.
Runtime Error Occurs while the program is executing.
Logical Error The program runs but produces incorrect results.
