1. Computer Systems & Programming Fundamentals

A computer is an electronic device that processes raw data into meaningful information. It combines hardware (physical parts) and software (programs that give instructions).

The Programming Hierarchy

  • Low‑Level Languages – close to machine hardware.
    • Machine Code: Binary (0s and 1s) – the only language the CPU understands directly.
    • Assembly: Uses mnemonics (e.g., MOV, ADD) – one step above machine.
  • High‑Level Languages – human‑readable, like English/math.
    • C, Java, Python: Must be translated into machine code by a compiler or interpreter.
🔁 The Compilation Process:
Source Code (.c) → Compiler → Object Code (.o) → Linker → Executable File (.exe or a.out)

Why C?

  • Structured, efficient, and gives low‑level access to memory.
  • Foundation for many modern languages (C++, Java, C#).
  • Widely used in systems programming, embedded devices, and operating systems.

2. Problem Solving & Flowcharting

Before writing code, plan your solution. Flowcharts visually represent the logic.

🔷 Standard Symbols

  • ⬤ Oval – Start / End
  • ▭ Parallelogram – Input / Output
  • ▬ Rectangle – Process (e.g., sum = a + b)
  • ◇ Diamond – Decision (condition / if‑else)

📋 Example: Sum of Two Numbers

  1. Start (Oval)
  2. Input num1, num2 (Parallelogram)
  3. Process sum = num1 + num2 (Rectangle)
  4. Output sum (Parallelogram)
  5. End (Oval)
💡 Algorithm = Step‑by‑step procedure. Flowcharts are just a graphical way to show the algorithm.

3. C Language Basics

C is a compiled, high‑level language. A C program starts execution from the main() function.

Basic Structure

// Preprocessor directive – includes standard I/O library
#include <stdio.h>

int main() {
    // Statements go here
    printf("Hello, World!\n");
    return 0;   // 0 means successful execution
}

Key Components

  • #include – pulls in header files (like stdio.h for I/O).
  • int main() – the entry point. Every C program must have one.
  • printf() – prints formatted output to the console.
  • return 0; – signals the OS that the program finished without errors.
⚠️ Note: C is case‑sensitive. main is not the same as Main.

Variables & Data Types

int age = 20;             // integer
float pi = 3.14159;        // floating‑point
char grade = 'A';           // single character (single quotes)
double big = 1.23456789;   // double precision float

Format Specifiers for printf() / scanf()

  • %d – int
  • %f – float
  • %c – char
  • %s – string (array of chars)
  • %lf – double
📥 Input Example:
scanf("%d", &age);  — the & gives the address of the variable.

4. Control Structures & Iteration

Programs can make decisions and repeat tasks.

Decision Making

if / else

if (score >= 60) {
    printf("Pass\n");
} else {
    printf("Fail\n");
}

switch statement

switch (day) {
    case 1: printf("Monday"); break;
    case 2: printf("Tuesday"); break;
    default: printf("Other day");
}

Loops (Iteration)

for loop – when you know the number of iterations

for (int i = 0; i < 5; i++) {
    printf("i = %d\n", i);
}

while loop – condition‑controlled

int n = 1;
while (n <= 5) {
    printf("%d ", n);
    n++;
}

do‑while – runs at least once

int x;
do {
    printf("Enter positive: ");
    scanf("%d", &x);
} while (x <= 0);
🔁 Common Mistake: Forgetting to update the loop variable leads to infinite loops.

Example: Sum of Two Numbers (Full Code)

#include <stdio.h>
int main() {
    int a, b, sum;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum = %d\n", sum);
    return 0;
}

5. Modular Programming – Functions & Arrays

Functions

Break large problems into smaller, reusable pieces.

// Function definition
int add(int x, int y) {
    return x + y;
}

int main() {
    int result = add(5, 3);
    printf("Result = %d\n", result);
    return 0;
}
📌 Benefits: Reusability, easier debugging, and cleaner code.

Arrays

Store multiple values of the same type in a single variable.

int numbers[5] = {10, 20, 30, 40, 50};
// Access: numbers[0] = 10, numbers[4] = 50

for (int i = 0; i < 5; i++) {
    printf("%d ", numbers[i]);
}

Array with Function Example

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}
⚠️ Caution: C does not check array bounds. Accessing numbers[10] may cause undefined behavior.

📚 Summary of Key Concepts

  • Compilation turns source code into machine code.
  • Flowcharts help visualise logic before coding.
  • Data types – int, float, char, double.
  • Control – if/else, switch, for, while, do‑while.
  • Functions – modularise code; main() is the entry.
  • Arrays – store lists; index starts at 0.
