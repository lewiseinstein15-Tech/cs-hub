Discrete Structures I

The mathematical foundation of Computer Science. This unit develops the logical thinking required for algorithms, programming, databases, artificial intelligence and computer systems.

01 · What is Discrete Mathematics?

Discrete mathematics is the branch of mathematics that studies objects that are separate, countable and have distinct values.

The word discrete means individual or separate. Computer systems work with information represented as separate pieces such as bits, characters and numbers.

Discrete example: Number of students: 40, 41, 42, 43 You cannot have: 40.5 students

Discrete mathematics focuses on structures that can be analysed step by step, making it extremely important in Computer Science.

02 · Discrete Mathematics vs Continuous Mathematics

Continuous mathematics studies values that can change smoothly and contain unlimited possible values.

Continuous: Temperature: 25°C 25.1°C 25.15°C 25.153°C

Discrete mathematics deals with separate values that can be counted.

Discrete: Number of computers: 1 2 3 4
Computer systems are mainly discrete because digital devices represent information using separate states such as 0 and 1.

03 · Why Computer Scientists Study Discrete Mathematics

Computer Science is built on mathematical thinking. Discrete structures provide the tools used to design, analyse and improve computer systems.

  • Algorithms use logical step-by-step processes.
  • Programming uses conditions and Boolean logic.
  • Databases organise information using sets and relations.
  • Networks are represented using graphs.
  • Artificial intelligence uses mathematical models.

04 · Introduction to Mathematical Logic

Logic is the study of correct reasoning. In Computer Science, logic allows machines to make decisions based on rules.

Programming languages use logical statements to control what a program does.

Example: IF password is correct THEN allow access ELSE deny access

The basic building block of logic is called a proposition.

05 · Propositions and Truth Values

A proposition is a statement that can be classified as either True or False, but not both.

In mathematics and Computer Science, propositions are represented using letters such as P, Q and R.

P: Kenya is in Africa. Truth value: P = True Q: 10 is smaller than 3. Truth value: Q = False

Questions, commands and opinions are not propositions because they cannot be assigned a definite truth value.

"Close the door." Not a proposition. "What is your name?" Not a proposition.

06 · Logical Variables

A logical variable represents a proposition. It stores a truth value: True (T) or False (F).

Let: P = "The computer is connected to the internet." Possible values: P = T or P = F

Logical variables allow computer scientists to represent decisions mathematically.

07 · The NOT Operator (¬)

The NOT operator reverses the truth value of a proposition.

If: P = True Then: ¬P = False If: P = False Then: ¬P = True

False True

08 · The AND Operator (∧)

The AND operator combines two propositions. The result is True only when both propositions are True.

P = I have a laptop. Q = I have internet. P ∧ Q means: I have a laptop AND internet.

09 · The OR Operator (∨)

The OR operator produces True if at least one proposition is True.

P = I will study Programming. Q = I will study Mathematics. P ∨ Q means: I will study Programming OR Mathematics.
P Q P ∨ Q

True False True
False True True

10 · XOR Operator (Exclusive OR)

XOR means "one or the other, but not both".

It produces True when the two inputs are different.

11 · Implication Operator (→)

An implication is a logical statement that shows a condition and a result.

It is written as:

P → Q Meaning: "If P is true, then Q must happen."

Example:

P = I study hard. Q = I pass my exam. P → Q If I study hard, then I pass my exam.
P Q P → Q

True False False

12 · Biconditional Operator (↔)

A biconditional means that two statements are true together or false together.

P ↔ Q Means: "P if and only if Q."

Example:

A number is even if and only if it can be divided by 2 without a remainder.
P Q P ↔ Q
True True True
True False False
False True False
False False True

13 · Logical Equivalence

Two logical expressions are equivalent if they always produce the same truth value.

The symbol used is:

P ≡ Q

Example:

¬(P ∧ Q) ≡ ¬P ∨ ¬Q

This is one of De Morgan's Laws.

14 · Tautology, Contradiction and Contingency

Tautology

A tautology is a statement that is always True.

P ∨ ¬P A statement is either true or its opposite is true.

Contradiction

A contradiction is a statement that is always False.

P ∧ ¬P A statement cannot be both true and false.

Contingency

A contingency is a statement that can be True or False depending on the situation.

15 · Important Laws of Logic

Logical laws help simplify complex expressions.

Identity Laws: P ∧ True = P P ∨ False = P Domination Laws: P ∨ True = True P ∧ False = False Double Negation: ¬(¬P) = P

16 · De Morgan's Laws

De Morgan's Laws describe how NOT changes when applied to combined statements.

First Law: ¬(P ∧ Q) = ¬P ∨ ¬Q Second Law: ¬(P ∨ Q) = ¬P ∧ ¬Q

These laws are very important in programming, digital circuits and Boolean algebra.

17 · Introduction to Sets

A set is a well-defined collection of objects called elements or members.

Sets are represented using curly brackets { }.

A = {2,4,6,8} The elements of set A are: 2, 4, 6 and 8

If an object belongs to a set, we use the symbol:

∈ means "is an element of" Example: 4 ∈ A

If an object does not belong to a set, we use:

∉ means "is not an element of" Example: 5 ∉ A

18 · Ways of Representing Sets

Sets can be represented in different ways.

1. Roster Method

Elements are listed inside brackets.

A = {1,2,3,4,5}

2. Set Builder Method

A set is described using a rule.

A = {x | x is an even number less than 10} Meaning: A contains all even numbers below 10.

19 · Types of Sets

Empty Set

A set containing no elements.

A = { } or A = ∅

Finite Set

A set with a limited number of elements.

B = {1,2,3,4}

Infinite Set

A set that continues forever.

N = {1,2,3,4,5,...}

Universal Set

The universal set contains all objects being considered in a problem.

U = {1,2,3,4,5,6,7,8}

20 · Subsets

A set A is a subset of set B if every element in A is also contained in B.

The symbol used is:

A ⊆ B
A = {1,2} B = {1,2,3,4} Therefore: A ⊆ B

Every element of A appears in B.

21 · Power Sets

The power set is the collection of all possible subsets of a set.

A = {1,2} Subsets: ∅ {1} {2} {1,2} Power Set: P(A) = {∅,{1},{2},{1,2}}

If a set contains n elements, the number of subsets is:

Number of subsets = 2ⁿ

22 · Set Operations

Set operations allow us to combine and compare sets.

Union ( ∪ )

Union contains every element from both sets without repetition.

A = {1,2,3} B = {3,4,5} A ∪ B = {1,2,3,4,5}

Intersection ( ∩ )

Intersection contains only elements that appear in both sets.

A = {1,2,3} B = {3,4,5} A ∩ B = {3}

24 · Difference of Sets

The difference between sets shows elements that exist in one set but not in another.

A = {1,2,3,4} B = {3,4,5,6} A - B = {1,2} B - A = {5,6}

25 · Complement of a Set

The complement of a set contains all elements in the universal set that are not part of that set.

Universal Set: U = {1,2,3,4,5,6} A = {1,2,3} A' = {4,5,6}

26 · Relations

A relation describes how elements from one set are connected to elements of another set.

Set A: {1,2,3} Set B: {a,b,c} Relation: {(1,a),(2,b),(3,c)}

Relations are important in databases because they show connections between different pieces of information.

28 · Boolean Algebra

Boolean algebra is the mathematics used by digital computers. It works using two values:

  • 1 = True / ON
  • 0 = False / OFF
AND: 1 AND 1 = 1 OR: 1 OR 0 = 1 NOT: NOT 1 = 0

Boolean algebra is the foundation of processors, memory and digital circuits.

29 · Number Systems

Computers use different number systems to store and process information.

  • Decimal — Base 10
  • Binary — Base 2
  • Octal — Base 8
  • Hexadecimal — Base 16
